# Khatu Shyam Outstation Taxi Services Lucknow

## Website Overview

This is a professional, conversion-focused landing page for Khatu Shyam Outstation Taxi Services based in Lucknow, Uttar Pradesh. The website is designed to be simple, honest, and ready for immediate deployment with Google Ads traffic.

## Features

- **Single-Page Design**: Clean, focused landing page with all essential information
- **Mobile-First**: Responsive design with sticky call/WhatsApp buttons on mobile
- **WhatsApp Integration**: Primary booking method via WhatsApp with pre-filled messages
- **Click-to-Call**: Direct phone number links for instant booking
- **Professional Fleet Display**: Clear separation between owned and partner vehicles
- **Transparent Pricing**: Honest pricing with clear notes about additional charges
- **Anonymous Feedback**: Generic customer testimonials without fake ratings
- **Fast Loading**: Optimized HTML/CSS with no heavy frameworks

## Deployment Instructions

### Option 1: Simple Hosting (Recommended)

1. **Upload to any web hosting service**:
   - Netlify (Free): Drag and drop the entire folder
   - Vercel (Free): Connect to GitHub or drag and drop
   - GitHub Pages (Free): Push to repository and enable Pages
   - Traditional hosting: Upload via FTP to public_html folder

2. **Domain Setup**:
   - Point your domain to the hosting service
   - Recommended domain: khatushyamtaxi.com or similar

### Option 2: Quick Test Locally

1. Simply open `index.html` in any web browser
2. All functionality works without a server

### Option 3: Deploy to Netlify (Step-by-Step)

1. Go to [netlify.com](https://netlify.com)
2. Sign up for free account
3. Drag the entire project folder to Netlify
4. Your site will be live in seconds with a free subdomain
5. Optionally add your custom domain
