## üîß TECHNICAL DECISIONS

### Why No JavaScript?
- **Faster loading**: No script parsing
- **Better SEO**: Pure HTML is crawlable
- **No bugs**: Nothing to break
- **Universal compatibility**: Works everywhere

### Why No Backend?
- **No booking needed**: WhatsApp handles it
- **No data storage**: Privacy-friendly
- **No maintenance**: Set and forget
- **No security risks**: No database to hack

### Why Single Page?
- **Focused conversion**: No navigation distractions
- **Mobile-friendly**: Easy scrolling
- **Fast loading**: One HTTP request
- **Clear journey**: Top to bottom flow

### Why Vanilla CSS?
- **Full control**: No framework limitations
- **Smaller size**: Faster loading
- **No dependencies**: No CDN failures
- **Custom design**: Unique appearance
